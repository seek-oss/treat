# 🍬 gatsby-plugin-treat

## Setup

To use `treat` in a `gatsby` project, install `gatsby-plugin-treat` and add it to your `gatsby-config.js` file like this:

```
module.exports = {
  plugins: [`gatsby-plugin-treat`],
};
```
